## 目的 / Purpose

SRE運用で使用するRunbook（手順書）を集約するリポジトリです。  
インシデント対応、定常運用、障害復旧の手順を構造化し、誰でも再現可能な形で管理します。

This repository collects runbooks for SRE operations, including incident response, routine operations, and disaster recovery procedures.
